Proyecto 8: Cuadrícula de Imágenes

Diseño responsivo usando CSS Grid

Instrucciones del Proyecto

Este proyecto requiere crear un diseño de cuadrícula usando HTML y CSS. Se deben arreglar seis imágenes en un patrón de cuadrícula basado en la maqueta proporcionada. El objetivo principal es aprender y practicar la técnica de diseño CSS Grid, que es esencial para crear diseños web sensibles y flexibles.

Objetivos de Aprendizaje

  • Estructura de Cuadrícula: Comprender los fundamentos de CSS Grid, incluyendo la definición de contenedores de cuadrícula, filas, columnas y cómo colocar elementos dentro de la cuadrícula.
  • Diseño Responsivo: Aprender a hacer que el diseño de la cuadrícula sea responsivo para que se adapte a diferentes tamaños de pantalla. Esto puede implicar cambiar el número de columnas o el diseño de las imágenes según el ancho del viewport.
  • Alineación y Espaciado: Explorar cómo alinear elementos de la cuadrícula y gestionar el espacio entre ellos para lograr un diseño limpio y visualmente atractivo.
